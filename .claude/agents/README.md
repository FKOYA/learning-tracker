# .claude/agents/

Claude Code のサブエージェント（専用ペルソナ）定義を置くディレクトリ。

## 追加するとき

メインのアシスタントとは別に、独自のシステムプロンプト・利用可能ツール・モデルを持たせたい専門タスク（例: 専用のコードレビュー担当）ができたときに追加する。現時点ではこの規模のアプリでサブエージェントを分ける必要はなく、このディレクトリに定義はない。

## 形式

`.claude/agents/<agent-name>.md` として1ファイル1エージェント。YAML frontmatter でシステムプロンプト・利用可能ツール・使用モデルなどを定義する。
